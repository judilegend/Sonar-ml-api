# 🔊 Sonar ML API

Sonar classification API based on an SVM model — distinguishes a **Mine** from a **Rock**
using 60 sonar energy measurements from the UCI Sonar dataset.

## Architecture

```
sonar-ml-api/
├── app/
│   ├── main.py        ← FastAPI application: endpoints and routing
│   ├── model.py       ← model loading and inference logic
│   └── schemas.py     ← Pydantic request/response validation
├── ml/
│   └── train.py       ← SVM training script and model persistence
├── tests/
│   └── test_api.py    ← pytest test suite
├── models/            ← generated by train.py (gitignored)
│   └── sonar_model.pkl
├── Dockerfile         ← multi-stage build: trainer + api
├── docker-compose.yml
├── .github/workflows/
│   └── ci_cd.yml      ← GitHub Actions CI/CD pipeline
└── requirements.txt
```

## Quick Start

### Option A — Local development (no Docker)

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the model (generates models/sonar_model.pkl)
python ml/train.py

# 3. Run the API
uvicorn app.main:app --reload

# 4. Open the interactive docs
# http://localhost:8000/docs
```

### Option B — Docker Compose

```bash
# Build and start the API
docker compose up --build

# Train the model inside Docker (if needed)
docker compose run --rm trainer

# The API is available at http://localhost:8000
```

## Endpoints

| Method | URL              | Description                 |
| ------ | ---------------- | --------------------------- |
| GET    | `/health`        | API and model health status |
| GET    | `/model/info`    | Loaded model metadata       |
| POST   | `/predict`       | Single signal prediction    |
| POST   | `/predict/batch` | Batch predictions           |
| GET    | `/docs`          | Swagger UI documentation    |

## CI/CD Pipeline

The `.github/workflows/ci_cd.yml` workflow defines 3 jobs:

1. **Test** — lint with flake8 and run pytest (on every push and PR)
2. **Build** — build the multi-stage Docker image (on push to main)
3. **Publish** — push the image to Docker Hub (on push to main)
